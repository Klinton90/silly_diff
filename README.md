# silly_diff
Allows to compare XML 2 XML or XML 2 DB objects.

More information can be found in Wiki.

## Requirements:
* Groovy-all 2.4.4 (at least it has been tested/developed around that version)
* Slf4j-api 1.7.12
* Gradle 2.3
